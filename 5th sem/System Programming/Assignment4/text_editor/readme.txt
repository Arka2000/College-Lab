Commands:
m / menu : gui menu
i / insert: insert mode
s / select: select mode
[esc] / [tab]: normal mode
c(select mode): copy
x(select mode): cut
p / paste: paste
find str: find str
replace str1 str2: replace str1 with str2
save: save file
new file: open new file
saveas str: save file as str
q / quit: quit
